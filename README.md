Purpose

The  program is a Point of Sales(POS) system developed for a Best Buy Retail Store. The purpose of the system is to digitize the sale process, reduce manual error and improve efficiency when processing customer purchase

This system will allow the cashier to :

Add items to a shopping cart 
Remove items from the shopping cart 
View the contents of the cart 
Calculate the total bill inclusive of tax and discounts (if any)
process payment 
Generate a receipt 

How to Run

To run this program the user would need to install the following software

1. Python
2. Pandas

after which save the code in a py.file and the run the program. Once the program is loaded, the user would then follow the on-screen instructions

Assumption

User will enter the correct product name or number 
User enters valid numeric input for quantity and payment 
CSV file is available for storing inventory  

Limitation
Limited to one customer at time
